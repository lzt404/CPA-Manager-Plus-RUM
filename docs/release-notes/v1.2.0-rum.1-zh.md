# CPA Manager Plus RUM v1.2.0-rum.1

这是基于上游 CPA Manager Plus v1.2.0 的 RUM 定制发布版本。

## 上游同步

- 同步 CPA Manager Plus v1.2.0 上游更新。
- 保留 RUM 版本的 API Key 访问范围控制能力。

## RUM 功能

- 每个客户端 API Key 可限制其可访问的凭证和 API provider。
- 可视化配置编辑器支持编辑 `allowed-auth-indexes` 和 `allowed-auth-ids`。
- 兼容旧版 `api-key-access-rules`，保存时迁移为 `api-keys` 内联规则。
- Manager Server 代理白名单包含 `/api-key-access-rules` 管理路径。

## 发布包

- Native 压缩包文件名改为 `cpa-manager-plus-rum_<version>_<os>_<arch>` 开头。
- 压缩包内不再包含顶层目录，解压后即可直接运行可执行文件。
- 压缩包内的可执行文件名保持上游现状：`cpa-manager-plus` / `cpa-manager-plus.exe`。
