# CPA Manager Plus RUM v1.1.2

这是 CPA Manager Plus RUM 的首个正式 Latest 发行版，基于上游 CPA Manager Plus v1.1.1，并加入 RUM 定制访问控制能力。

## 新增

- 为每个客户端 API Key 增加访问范围控制，可限制其可访问的凭证和 API provider。
- 支持在可视化配置编辑器中编辑 `allowed-auth-indexes` 和 `allowed-auth-ids`。
- API Key 编辑器可从现有 auth 文件和 provider 配置中选择可访问项，并支持批量加入。
- 兼容旧版 `api-key-access-rules` 配置，保存时迁移为 `api-keys` 内联规则。
- Manager Server 代理白名单加入 `/api-key-access-rules` 管理路径。

## 发布

- Release workflow 已适配 RUM 仓库，并会把正式版本标记为 GitHub Latest。
- GitHub Container Registry 镜像发布到 `ghcr.io/lzt404/cpa-manager-plus-rum`。
- DockerHub 发布目标为 `lzt404/cpa-manager-plus-rum`，未配置 DockerHub secrets 时会自动跳过。

## 验证

- Frontend、Manager Server、Docker Build 均已在 GitHub Actions 通过。
- 前端类型检查、lint、测试和生产构建均已通过。
